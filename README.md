# WeatherService

Sample apps that call the [OpenWeatherMap Current Weather API](https://openweathermap.org/current).  You'll need to sign up for an account in order to obtain an API key.

**Projects**

| Project | What it is |
|--------|-------------|
| **SimpleConsoleUI** | Minimal `HttpClient` demo to demonstrate simplicity from the console.|
| **WeatherConsoleUI** | More sophisticated console app using Microsoft generic host for logging, configuration and dependency injection. |
| **WeatherServiceApp** | Full Blazor Web App demo. |
